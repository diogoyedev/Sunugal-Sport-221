# 🏋️ Sunugal Sport 221

> *"L'excellence physique au cœur de la Teranga."*

Site web vitrine d'une salle de sport sénégalaise, réalisé dans le cadre d'un projet de fin de premier semestre.

🔗 **[Voir le site en ligne](https://sunugal-sport-221-15pig3vh9-diogoyedevs-projects.vercel.app/)**

---

## 👥 Équipe

| Rôle | Responsabilité |
|------|---------------|
| 🎨 Designer | Maquette Figma, identité visuelle |
| 💻 Développeur | Intégration HTML/CSS, déploiement |

---

## 📄 Pages

- **Accueil** — Hero section, section "Pourquoi nous ?", stats
- **Nos Coachs** — Présentation des 6 coachs avec cards
- **Programme** — Les programmes disponibles
- **Tarifs** — 3 formules : Mensuel, Trimestriel, Annuel
- **Inscription** — Formulaire de contact
- **Contact** — Informations de contact

---

## 🛠️ Technologies utilisées

- **HTML5** — Structure des pages
- **CSS3** — Style, animations, responsive
  - Flexbox & CSS Grid
  - Media Queries
  - Animations & transitions
  - CSS custom properties
- **Iconify** — Icônes
- **inter** — Typographie (la police complete)

---

## ✨ Fonctionnalités

- Design **dark mode** avec thème orange
- Navigation active par page
- Cards avec effets **hover** (scale + glow)
- Formulaire d'inscription multi-champs
- Layout **responsive** (mobile, tablette, desktop), (meme si le format responsif mobile reste a desiser <0-0>!)
- Footer complet avec liens rapides et réseaux sociaux
